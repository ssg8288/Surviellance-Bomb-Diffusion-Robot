* Control A Robotic Arm Wirelessly Using a Glove With Flex and Motion Sensors

---

* NRF24 Wiring:

* VCC - 3.3V (or use an adapter for 5V, like I did)

* GND - GND

* CE - D8

* CSN - D10

* SCK - D13

* MOSI - D11

* MISO - D12

* IRQ - N/C

---

* MPU6050 Wiring:

* VCC - 5V

* GND - GND

* SDA - A4

* SCL - A5

* INT - D2

Make Sure you used these following header files in your arduino code.
